# RuStream

[![Скачать APK](https://img.shields.io/badge/⬇_Скачать_APK-latest-6750A4?style=for-the-badge)](https://github.com/Komsomol39/rustream/releases/latest/download/app-debug.apk)

Android-приложение «всё в одном» для поиска, скачивания и воспроизведения медиа: торрент-трекеры, онлайн-сервисы (YouTube, SoundCloud, VK и сотни других через yt-dlp), встроенные плееры видео и музыки с фоновым воспроизведением.

> Личный проект. APK собирается автоматически через GitHub Actions.

## Возможности

**Поиск и загрузка**
- Торрент-поиск по нескольким трекерам сразу: YTS, RuTor, Kinozal, RuTracker, NNM-Club
- Авторизация на приватных трекерах через встроенный WebView (куки сохраняются)
- Скачивание magnet-ссылок и `.torrent`-файлов на встроенном движке libtorrent4j
- Выбор отдельных файлов внутри раздачи (можно качать только нужные серии)
- Онлайн-загрузки через NewPipe (поиск) + yt-dlp (скачивание) — видео в лучшем качестве или аудио в mp3
- Скачивание по прямой ссылке с сотен сайтов

**Плеер и библиотека**
- Видеоплеер на Media3/ExoPlayer с FFmpeg-декодером (DTS/AC3 и прочие дорожки), запоминает позицию просмотра
- Музыкальный плеер с фоновым воспроизведением, управлением из шторки и с экрана блокировки
- Перемешивание, повтор плейлиста/трека, «играть всё»
- Группировка музыки по исполнителям с объединением псевдонимов
- Превью-миниатюры для видео
- Подключение произвольных папок устройства как источников медиа

## Технологии

- **Язык:** Kotlin, Jetpack Compose
- **Архитектура:** MVVM, Hilt (DI), DataStore
- **Торренты:** libtorrent4j (posix disk I/O)
- **Медиа:** AndroidX Media3 (ExoPlayer, MediaSession) + Jellyfin FFmpeg decoder
- **Онлайн:** NewPipeExtractor, youtubedl-android (yt-dlp + FFmpeg)
- **Сеть:** OkHttp, Jsoup

## Сборка

Приложение собирается в облаке через GitHub Actions (workflow **Build APK**):

1. Внести изменения в код
2. Обновить файл `build-trigger` в корне репозитория
3. Дождаться сборки (~5 минут) → артефакт `RuStream-debug` со скомпилированным APK

Локальная сборка стандартная для Android-проекта:

```bash
./gradlew assembleDebug
```

Требуется JDK 17 и Android SDK. Собирается только под `arm64-v8a`.
